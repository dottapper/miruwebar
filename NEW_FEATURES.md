# 🎯 今日追加された新機能

## ✅ プロジェクト一覧画面の変更点

### 1. 新しい「ARで見る」ボタン
- 各プロジェクトカードの右下に紫色のボタンが追加
- ボタンテキスト: "ARで見る"
- 3Dキューブアイコン付き

### 2. 新しいサイドメニュー項目
- サイドメニューに「QRコードスキャン」が追加
- QRコードアイコン付き

## ✅ 動作確認方法

### ステップ1: ブラウザで https://localhost:3000/ にアクセス
### ステップ2: ハードリロード (Cmd+Shift+R)
### ステップ3: プロジェクト一覧で以下を確認
- サンプルプロジェクトが表示される
- 各カードに紫色の「ARで見る」ボタンがある
- サイドメニューに「QRコードスキャン」がある

## 🚨 変更が見えない場合
1. シークレット/プライベートモードで開く
2. F12 → Network → "Disable cache"
3. ページを再読み込み
