# Legacy Files

このディレクトリには、現在のシステムで使用されなくなった古い実装ファイルが保存されています。

## 移動されたファイル

### 古いローディングシステム

- `loadingManager.js` - 旧ローディング画面管理システム（統一ローディングシステムに移行済み）
- `loading-screen.css` - 旧ローディング画面 CSS（統一ローディングシステムに移行済み）

### 未使用の AR 実装

- `webxr-ar.js` - WebXR ベースのマーカーレス AR 実装（現在はマーカー型 AR のみサポート）

### バックアップファイル

- `project.json.backup.*` - プロジェクトファイルの自動バックアップ

## 注意事項

- これらのファイルは現在のシステムでは使用されていません
- 必要に応じて参照用として保持されています
- 削除しても現在のシステムの動作に影響はありません

## 復旧方法

もしこれらのファイルが必要になった場合は、適切なディレクトリに移動し直してください：

```bash
# 例：loadingManager.jsを復旧する場合
mv legacy/loadingManager.js src/utils/
```
